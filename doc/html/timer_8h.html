◆ TIMER_PRESCALER

#define TIMER_PRESCALER   ((uint32_t)8)

Formula for calculating the corresponding prescaler value for a desired timer resolution: PR = Tres * fper - 1 with fper = 72 MHz and Tres = 125 ns PR = 125 * 10^-9 * 72 * 10^6 - 1 = 8

The PWM signal will be LOW at the start of a PWM period and HIGH once it reaches the duty cycle, defined by MR0 - MR2, until the end of the period. Sets the the resolution of the timer to 125 ns.

Function TIMER_init() initializes a 16-bit/32-bit PWM.

Parameters
timerindicates, which timer will be initialized.
prescaler_valdefines the prescaler value.
mcr_valvalue to which the MCR register shall be set to.
match_valsdetermines the values for the match registers 1, 2 and 3.
irq_priodetermines the interrupt priority. If no IRQ via mcr_val is selected, this is ignored.
ctcr_valdetermines the timer mode and the count input select.
pwmc_valdetermines, which match register shall be used as PWM output.
Returns
Bool
Return values
TRUEconfiguration success
FALSEinvalid parameter(s) provided
